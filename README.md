# Autogrid

**Helps to setup grid size faster**

And also change background images
---

## 📦 What Does This Module Do?

Custom Target Icon allows multiple target manipulations:

- **Setup grid size** (from the base tile)
- **Change background** (uses keywords from the background file name)

## 🎯 Installation

### Method 1: Module Browser (Currently unavailable, will try to add later)
1. In Foundry VTT, go to **Add-on Modules**
2. Click **Install Module**
3. Search for **"Autogrid"**
4. Click **Install**
5. Enable it in your world

### Method 2: Manifest URL
1. Copy this URL: `[manifest URL](https://github.com/olleneri/autogrid/blob/main/module.json)`
2. In Foundry, go to **Add-on Modules → Install Module**
3. Paste the URL in **Manifest URL**
4. Click **Install**

## 🚀 How to Use

### Step 1: Open module settings
Open module settings.

### Step 2: Setup everything 
1. When Enable Background Toggle is ON pressing B will try to change background image from **...grid...** to **...no grid...**
2. Light/Dark Expressions when pressed N module will change background image from **...night...** to **...day...** or **...light...** to **...dark...** and back. You can use different keys in background image titles.
3. HQ/LQ Expressions is same as Light/Dark Expressions but for Low Quality/High Quality images.
4. Custom Toggle Groups - same as Light/Dark Expressions but you can use multiple background images to show evolution of a map. 

### Step 3: Save
Click **"Save"** at the bottom of the window.

### Step 4: Configurating grid
Press M and draw a square. It might not work from the first try.
